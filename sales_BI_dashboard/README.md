# BI-дашборд продаж и финансовых показателей (NDA)

## Описание проекта
Разработан BI-дашборд для комплексного анализа прибыльности товаров маркетплейса.
Решение объединяет операционную и финансовую аналитику и фактически выполняет роль управленческого отчёта по продажам.
В связи с NDA реальные данные и ссылка на production-дашборд не публикуются.
Репозиторий демонстрирует структуру метрик, бизнес-логику расчётов и архитектурный подход к построению BI-решения.

## Бизнес-цели
Дашборд позволяет ответить на ключевые вопросы бизнеса:
какая доля заказов конвертируется в выкупы
как возвраты влияют на чистые продажи и прибыль
какая валовая и маржинальная прибыль по каждому артикулу
насколько эффективно расходуется рекламный бюджет
какие статьи затрат оказывают наибольшее давление на маржинальность

## Ключевая логика расчётов

Основная таблица
Основная таблица дашборда формируется на основе данных по заказам.
Правила расчётов:
Продажи и возвраты учитываются в месяце создания заказа, а не в момент фактического выкупа.
Такой подход позволяет корректно оценивать текущую воронку продаж и не завышать показатели фактических выкупов.
Себестоимость и валовая прибыль
Себестоимость товаров формируется на основе данных о поставках:
данные загружаются через API маркетплейса и дополняются вручную в базе по каждой партии.
на основе себестоимости рассчитываются валовая прибыль и рентабельность по валовой прибыли.

На базе себестоимости рассчитываются:
валовая прибыль
рентабельность по валовой прибыли
Расходы и маржинальная прибыль
В модели учтены ключевые статьи затрат:
рекламные расходы (привязаны к дате заказа)
комиссия маркетплейса
эквайринг
логистика (прямая и обратная)
хранение
Финальные показатели:
маржинальная прибыль
рентабельность по маржинальной прибыли

## Структура дашборда
1. Динамика ключевых финансовых показателей
Линейные графики с временной динамикой:
Доля рекламных расходов: ДРР от заказов, ДРР от выкупов
Прибыль: валовая и маржинальная прибыль
Рентабльность: по валовой прибыли и по маржинальной прибыли
% выкупа 
Назначение: быстрая оценка трендов и выявление аномалий.
2. Отчёт о доходах, расходах и прибыльности (аналог ОПУ)
- Таблица, содержащая структуру доходов и расходов,
валовую и маржинальную прибыль
показатели рентабельности
- Таблица с динамикой основных показателей:
текущий месяц (факт / прогноз)
предыдущий месяц
изменение month-over-month (MoM)
- Дополнительно представлена кольцевая диаграмма структуры расходов для быстрого выявления ключевых драйверов затрат.
4. Детализация заказов
Табличный аналитический слой с разбивкой по артикулам:
заказы, выкупы, возвраты
цена продажи для селлера
расходы
Используется для:
поиска причин отклонений
анализа unit-экономики
подготовки управленческих решений
5. Поставки и себестоимость
Лист с аналитикой по товарным поставкам:
объёмы поставок
себестоимость по артикулам
Позволяет:
анализировать влияние себестоимости на прибыль
обеспечивать корректность финансовых расчётов
